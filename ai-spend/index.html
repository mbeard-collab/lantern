Option A — Python (any machine): Open Terminal in your Downloads folder and run: python3 -m http.server 8765 then open http://localhost:8765/govspend_unified_dashboard.html
Option B — Node: npx serve . then open the URL it prints.
Option C — VS Code: Install the Live Server extension → right-click this file → Open with Live Server.
CSV uploads work fine from file:// — only the Admin API pull requires HTTP.
